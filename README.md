# 照片备份工具

这是一个简单的照片备份工具，用于将照片文件从源目录备份到指定的目标目录，支持处理
**JPEG** 和各类 **RAW** 格式照片。

## 功能特点

-   支持选择源目录和目标目录（JPEG 和 RAW 文件）
-   支持选择 RAW 文件格式（多品牌、多机型）
-   支持在文件重复时选择处理方式
-   支持保留文件夹结构
-   提供进度条显示备份进度
-   提供完整日志记录功能

## 界面说明

主界面包含以下区域：

### 1. 文件夹选择区域

-   源目录选择框与浏览按钮\
-   JPEG 目标目录选择框与浏览按钮\
-   RAW 目标目录选择框与浏览按钮

### 2. 配置区域

-   RAW 文件格式选择复选框（支持多种相机格式）\
-   重复文件处理方式下拉框

### 3. 控制区域

-   **开始备份** 按钮

### 4. 进度显示区域

-   进度条\
-   进度百分比标签\
-   操作日志列表框

## 运行环境

-   Windows 7 / 8 / 10 / 11\
-   .NET Framework 4.x 或 .NET 6+\
-   建议以 **管理员权限** 运行

## 使用方法

1.  打开程序\
2.  点击 **"选择源目录"** 按钮选择需要备份的照片目录\
3.  点击 **"选择 JPEG 目标目录"** 按钮设置 JPEG 文件存储位置\
4.  点击 **"选择 RAW 目标目录"** 按钮设置 RAW 文件存储位置\
5.  选择需要备份的 RAW 文件格式\
6.  选择文件重复时的处理方式\
7.  点击 **"开始备份"**\
8.  通过进度条和日志查看备份进度

## 注意事项

-   确保目标目录有足够的存储空间\
-   备份过程中请勿关闭程序\
-   网络路径备份需要稳定的网络连接\
-   建议先备份少量文件以验证配置正确性
